# Fork Lineage

Hamr started as a Pi fork and still carries that history in the codebase and package layout. This site documents the current Hamr behavior, names, and commands.

## Current Names

- CLI package: `@hamr/coding-agent`
- Library packages: `@hamr/agent`, `@hamr/ai`, `@hamr/tui`
- Repository: `skaft-software/hamr`

## What To Trust

- Use this site for current Hamr commands and runtime behavior.
- Treat historical Pi-era material in `packages/coding-agent/docs/` as attribution and background, not the source of truth.
- Expect the runtime architecture to keep changing. The docs track the current state, not a frozen spec.

## Why This Exists

Hamr kept the useful parts of the Pi fork and extended them. Some names still reflect the origin, but the public docs should describe Hamr as it exists now.
