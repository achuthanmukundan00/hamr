# Hamr: The Coding Agent for Local Models

Hamr is the de-facto coding agent for local models, built on top of Pi as an agent toolkit. Designed to help little models survive and big models breathe.

This site documents Hamr's behavior, commands, and architecture.

## Packages

- CLI: `@hamr/coding-agent`
- Libraries: `@hamr/agent`, `@hamr/ai`, `@hamr/tui`
- Repository: `skaft-software/hamr`

## What To Trust

- This site documents Hamr's current commands and runtime behavior.
- Package names reflect Pi's origins, but this site describes Hamr as it exists today.
- The runtime architecture is still evolving. These docs track the current state, not a frozen spec.

## Why This Exists

Hamr inherits Pi's strengths as an agent toolkit and extends them for local model coding tasks. This site focuses on Hamr as the primary coding agent for local models.
